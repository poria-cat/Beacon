# Beacon

Beacon is simulation for Dfinity Random-Beacon.

### Feature

* Threshold Relay
* Verifiable Random Function
* Distributed Key Generation
* Block 
* Mini Blockchain

### Run

* `npm install` or use `yarn` to install modules
* `node index.js`

Then, Beacon will working in your terminal
